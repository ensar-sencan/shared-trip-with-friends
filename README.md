# 💸 SplitFlow

https://www.loom.com/share/e0eed8e3c0484b7a88b1cfdcd4b2a3c2

**Calculate group expenses, pay via Stellar blockchain.**

SplitFlow is a modern web application that allows you to split costs fairly in trips, events, or shared expenses with friends and make secure payments via the Stellar blockchain.

[![Freighter](https://img.shields.io/badge/Freighter-Wallet-10B981?style=flat)](https://www.freighter.app)

## ✨ Features

### 🔐 Blockchain Integration
- Secure connection with **Freighter Wallet**
- Flexible payment options with **Stellar Path Payment**
- Real blockchain transactions on **Testnet**
- All payments are transparent and verifiable on the blockchain

### 📊 Smart Calculation
- Automatic debt calculation algorithm
- Optimal payment plan with a minimum number of transfers
- Real-time balance tracking
- Detailed debt graph visualization

### 🎯 User-Friendly Interface
- Modern and professional design
- Instant notifications with live Horizon SSE stream
- Easy group invitation via QR code
- Karma scoring system
- Category-based expense analysis

### 🚀 Demo Mode
- Rich demo environment with 5 mock users
- 14 sample expense scenarios
- Ability to test real Freighter payments
- Mock address support for local testing

## 🛠️ Technologies

### Frontend
- **React 18** + **TypeScript**
- **Vite** - Fast build tool
- **React Router** - Page routing
- Modern CSS3 animations

### Blockchain
- **Stellar SDK** - Blockchain integration
- **Soroban** - Smart contract support
- **Freighter API** - Wallet connection
- **Horizon API** - Real-time transaction tracking

### Algorithms
- Minimum transfer calculation
- Debt graph optimization
- Path payment routing
- Karma scoring system
